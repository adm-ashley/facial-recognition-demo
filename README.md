# Facial Recognition Demo

## Instructions

Follow these instructions to get a copy of the project running on your local machine.

### Prerequisites

This project requires [Node.js](https://nodejs.org/en/download/package-manager/) and [npm](https://www.npmjs.com/get-npm) to be installed.

```sh
brew install node
```

```sh
npm install npm@latest -g
```

### Installing

1. Clone this repository and navigate inside this project's folder ([facial-recognition demo](https://github.com/adm-ashley/facial-recognition-demo.git)).
   ```sh
   git clone https://github.com/adm-ashley/facial-recognition-demo.git
   ```
2. Install npm packages.
   ```sh
   npm install --save express
   ```
3. Run a local server.
   ```sh
   node server.js
   ```
4. Access the project at http://localhost:3000/ in a browser (works best in Chrome).
